# Patient Management Frontend

React application for managing and reviewing patient medical documents.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- **Patient Selection Page**: Overview of all patients with executive summary
- **Patient Information Page**: Detailed view with color-coded document highlights, timeline, and filtering

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router
